## Shortcuts
> Super = Windows, Control in Macos

| Keys | Action |
| ---- | ------ |
| super + (1-0) | change desktop
| Super + return | kitty
| Super + w | web browser
| super + e | file explorer
| super + space | app explorer
| super + q | quit app
| super + s | screenshot
| super + shitft + L | logout
